# Strings

## What is a String

A sequence of characters.

```python
s = "hello"
```

## Basic Operations
| Operation | Complexity |
| --- | --- |
| Access | O(1) |
| Concatenation | O(n) |
| Search | O(n) |

## Important Concepts

### Palindrome

String that reads same forward and backward.

**Example:**
- `racecar`

### Anagram

Two strings with same characters.

**Example:**
- `listen`
- `silent`

## String Matching Algorithms
- KMP Algorithm
- Rabin Karp
- Z Algorithm

## Common Problems
- Longest substring without repeating characters 
- Valid palindrome 
- Group anagrams
