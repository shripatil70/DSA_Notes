# What is Hashing

Mapping data to a fixed index using a hash function.

## Example:

```python
hashmap = {}
```

## Hash Table Components
- **Key**
- **Value**
- **Hash Function**
- **Bucket**

## Advantages
- Fast lookup
- Average complexity O(1)

## Collision Handling
- Chaining
- Open addressing

## Common Problems
- Two Sum
- Frequency count
- Top K frequent elements
