What is Time Complexity

Time complexity measures how runtime grows with input size.

Common complexities:

| Complexity | Meaning      |
| ---------- | ------------ |
| O(1)       | Constant     |
| O(log n)   | Logarithmic  |
| O(n)       | Linear       |
| O(n log n) | Linearithmic |
| O(n²)      | Quadratic    |


Examples:

| Operation     | Complexity |
| ------------- | ---------- |
| Array access  | O(1)       |
| Linear search | O(n)       |
| Binary search | O(log n)   |


Interview tip:
Always mention time and space complexity after writing code.
