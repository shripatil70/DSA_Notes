# Bit Manipulation

Works at binary level.
Bit manipulation is a fundamental programming technique that involves operating on individual bits within data to optimize performance and memory usage in data structures and algorithms. It leverages bitwise operators, which are processed efficiently by the CPU, often in a single clock cycle.

## Operators:

| Operator | Meaning |
| -------- | --------|
| &        | AND     |
| |        | OR      |
| ^        | XOR     |
| ~        | NOT     |
| <<, >>  | Left/Right Shifts for fast multiplication or division by powers of 2 |

Learn Bit Manipulation: [https://leetcode.com/discuss/post/136585/a-summary-how-to-use-bit-manipulation-to-ty85/](https://leetcode.com/discuss/post/136585/a-summary-how-to-use-bit-manipulation-to-ty85/)

## Example:

```
5 = 101
3 = 011
AND = 001
```
